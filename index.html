    rSalvador: An R Tool for the Luria-Delbruck Fluctuation Assay

    Documentation

    Follow the guide for installation instructions and a basic tutorial. Windows users can download the mentioned distribution file from here. Linux users can download the tarball from here. The rSalvador manual is also available.

    Installing from source

    R CMD check rsalvador
R CMD INSTALL rsalvador

    Building an installable archive

    R CMD check rsalvador
R CMD INSTALL --build rsalvador
